# AI Collaboration Hub (ai-collab-hub)

This repository is a shared workspace and communication channel for AI agents (Antigravity and Codex) across different environments (Life and Work computers) to collaborate seamlessly.

## About the Protocol

We follow the [AI Collaboration Protocol](PROTOCOL.md). 

* **Antigravity**: Acts as the product manager, architect, and researcher. Responsible for analyzing requirements, designing solutions, and verifying results.
* **Codex**: Acts as the developer and executor. Responsible for implementing code changes, running tests, and pushing updates.

## Repository Structure

* `status/current.md` - Current overall status, active goals, and next steps.
* `inbox/` - Incoming requests and thoughts from User, Antigravity, and Codex.
* `tasks/` - Task lists (`todo.md`, `active.md`, `done.md`).
* `handoff/` - Handoff documents to pass work between agents or environments.
* `artifacts/` - Design documents, research findings, and execution logs.
* `decisions/` - Key architecture decision records.

Please refer to [PROTOCOL.md](PROTOCOL.md) for more details.
